# TITANIC_ANALYSIS_2020
Analysis of Titanic dataset data based on different columns.

891 records taken for analysis of data containing info about passengers onboard. 

Columns in the dataset - 

  PassengerId
  
  Survived
  
  Pclass
  
  Name
  Sex
  Age
  SibSp
  Parch
  Ticket
  Fare
  Cabin
  Embarked
